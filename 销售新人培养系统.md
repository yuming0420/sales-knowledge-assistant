# 何总销售新人培养系统

> 基于何总蒸馏知识库 | 山东朱氏药业 TOB 销售专用 | 8 周从零到独立开发客户

---

## 培养理念

```
训练控制意愿 → 工具控制行为 → 流程控制结果
     ↓               ↓               ↓
  让他想干          让他会干        让他出结果
```

**核心信条**：销售不是天赋，是体系。把猪放进加工厂，出来就是火腿肠。

| 阶段 | 周期 | 主题 | 目标 |

## P2 · 方法论筑基（第2周）

> 目标：七步销售法背得滚瓜烂熟，每一步都能用自己的话讲出核心+做法。

### 每日节奏

| 时段 | 内容 |
|------|------|
| 8:30-9:00 | 晨会：当日那一步的讲解 + 金句背诵 |
| 9:00-12:00 | 跟师傅旁听电话/拜访 |
| 14:00-15:00 | AI 助手深度练习当日步骤 |
| 15:00-17:30 | 模拟训练或继续跟学 |
| 17:30-18:00 | 夕会：当日心得 + 明日预习 |

### Day 1 — 第一步：计划与活动

**核心**：明确目标客户和开发方式。

**AI 练习**：
```
问：我是朱氏药业销售，想在济南开发3家药厂客户，帮我做一份计划与活动方案
```

**今日实操**：在天眼查上找出10家目标药厂，核实法人信息。

---

### Day 2 — 第二步：接触前准备

**核心**：准备好话术、工具、心态。

**AI 练习**：
```
问：我要去拜访一家药厂的总经理，帮我准备话术、工具清单和心态调整
```

**今日实操**：制作一张A4纸的"一页纸公司介绍"。

---

### Day 3 — 第三步：接触说明

**核心**：1分钟内吸引客户。

**AI 练习**：
```
问：给药厂老板打第一个电话，话术怎么写？目的只是加微信
```

**今日实操**：模拟给师傅打"第一个电话"，师傅评分。

---

### Day 4 — 第四步：追踪

**核心**：要么成为客户，要么让他记住你，要么有需求时第一时间找到你。

**AI 练习**：
```
问：加了药厂采购微信三天没回我，接下来怎么追踪？
```

**今日实操**：写3条不同场景的追踪短信，师傅审核后存档。

---

### Day 5 — 第五步：促成

**核心**：让客户占便宜。

**AI 练习**：
```
问：客户试样后说"再考虑考虑"，怎么逼单促成？
```

**今日实操**：列出朱氏药业能给客户的3种"便宜"（赠样、锁价、技术支持）。

---

### Day 6 — 第六步 + 第七步：售后服务 + 转介绍

**核心**：销售从售后服务开始；所有单方面索取最终都会失败。

**AI 练习**：
```
问：客户成交后怎么维护？怎么让老客户给我介绍新客户？
```

**今日实操**：写一份成交客户30天维护计划。

---

## P3 · 客户开发实战（第3-4周）

> 目标：从模拟到实战，真正开始打电话、加微信、寄样品。

### 每日铁律

```
上午 10:00-11:30 → 黄金开发电话时间（目标：打30个，加3个微信）
下午 15:30-17:30 → 二次电话窗口 + 追踪已加微信的客户
晚上 20:00-21:00 → 整理客户分类表（ABCD），规划明日追踪
```

### 第3周 · 客户开发周

| 日 | 重点 |
|----|------|
| Day 1-2 | 数据筛选：天眼查/企查查找出50家目标药厂，支付宝核实法人手机 |
| Day 3-5 | 实战打电话：以副总经理身份，目标加10个微信 |
| Day 6 | 寄样品：从已加微信中筛选3个意向客户，寄出样品 |
| Day 7 | 周复盘：统计电话数/微信数/样品数，师傅点评 |

**AI 助手每日必问**：
```
问：今天打了30个电话只加了2个微信，怎么提高？
问：客户说"发个资料看看"然后不回了，怎么追？
```

### 第4周 · 客户追踪周

| 日 | 重点 |
|----|------|
| Day 1-2 | 对已寄样客户进行追踪：确认收货、询问测试进度 |
| Day 3-4 | 对新加微信但未寄样的客户分类ABCD |
| Day 5-6 | 重点追踪A类客户，尝试约电话或拜访 |
| Day 7 | 周复盘：追踪转化率 |

**追踪铁律**：
- 样品寄出当天必须让客户知道已送达
- 超过3天未回复 → 打电话
- 超过1周未测试 → 重新激活（新品/行业资讯/假日礼品）

## P4 · 成交与维护（第5-6周）

> 目标：完成首单成交，建立售后服务SOP。

### 促成训练

**逼单四步法**：
1. 先确认样品测试结果 → "李总，样品测下来效果怎么样？"
2. 制造紧迫感 → "最近旺季产能紧张，确认合作的话我提前帮您排产"
3. 给出让利方案 → "首次合作我给您申请了锁价一年的优惠"
4. 降低决策门槛 → "先试一小批量，不满意随时调整"

**AI 每日练习**：
```
问：客户说"价格比别家贵"，怎么回应并促成？
问：客户一直拖，说"再等等"，怎么逼单？
```

### 售后服务SOP

成交后第1天：发送合作确认函 + 生产排期通知
成交后第3天：主动告知物流信息
成交后第7天：电话回访使用情况
成交后第30天：上门拜访或寄礼品
每月固定：发送行业资讯 + 市场动态

## AI 助手使用指南

新人在任何环节遇到问题，打开 Web 问答台或 CLI：

```bash
# 每天至少问1个实战问题
python sales_assistant.py "你的问题"
```

**高频场景速查**：

| 场景 | 问法 |
|------|------|
| 话术不会写 | "帮我写一段给药厂采购的跟进话术" |
| 客户不回 | "客户三天没回消息了怎么追？" |
| 价格异议 | "客户说我们价格贵怎么回？" |
| 竞品对比 | "客户说竞品比我们便宜怎么回应？" |
| 逼单促成 | "客户试样完了但不下单怎么逼？" |
| 客户分类 | "帮我分析这个客户属于ABCD哪类" |
| 心态崩了 | "今天被10个客户拒绝了，心态怎么调？" |
